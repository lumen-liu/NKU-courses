## Introduction

本仓库记录了南开求学四年中所有参与过的课程实验部分，包括：
- 自己独立完成的实验代码
- 课堂上老师展示的优秀示例
- 部分课程项目和作业

> ⚠️ **免责声明**：本人并非编程高手，部分代码质量可能不高，甚至存在"屎山代码"。请使用者自行鉴别和改进。如果发现任何问题，欢迎提PR或者通过邮箱联系我。
